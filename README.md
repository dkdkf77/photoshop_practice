# photoshop_practice
패스트캠퍼스 디자이너 과정 포토샵 그레이디언트 공부 


## 그레이디언트를 주면 강조 및 색감을 풍부하게 표현 하는 것이 가능하다.
### 포토샵 단축기 alt , mac은 option 은 복사에 관련된 단축기라고 한다.

- 사용 툴 : 포토샵 
- 레이어에 그레이디언트를 주었음.
- 그레이디언트로 노을 효과 및 해가 저물어 가는 효과를 주었음
- 픽셀 유동화를 사용하여 노을 번짐 효과를 주었음

- 그레이디언트로 만든 노을 사진
